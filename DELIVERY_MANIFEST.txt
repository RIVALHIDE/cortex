================================================================================
USAGE EXAMPLES
================================================================================

For End Users:
  $ cortex --language es install nginx
  $ export CORTEX_LANGUAGE=hi && cortex status
  $ cortex config language ja

For Developers:
  from cortex.i18n import get_translator
  t = get_translator('es')
  msg = t.get('install.success', package='nginx')
  # Returns: "nginx instalado exitosamente"

For Translators:
  1. cp cortex/translations/en.json cortex/translations/de.json
  2. Edit de.json and translate all values
  3. Add 'de': 'Deutsch' to SUPPORTED_LANGUAGES
  4. Test: cortex -L de install nginx --dry-run
  5. Submit PR
